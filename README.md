# Smart Complaint & Issue Tracking System

A full-stack web application designed to manage complaints efficiently with real-time tracking and admin control.

---

## 🚀 Features

- OTP-based user registration  
- Complaint submission with optional image upload  
- Real-time status tracking  
- Admin dashboard for complaint management  
- Search and filter functionality  
- Email notifications  
- Analytics dashboard (Chart.js)

---

## 🛠 Tech Stack

- Backend: Spring Boot (Java)
- Frontend: HTML, CSS, JavaScript
- Database: MySQL
- Tools: Chart.js, JavaMailSender
